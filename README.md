# Homebrew Tap for Studio CLI

```bash
brew install tokens-studio/studio-cli/studio
```

## About

[Studio CLI](https://tokens.studio) is a design tokens management tool for the command line.

## Other Installation Methods

```bash
# npm
npm install -g @tokens-studio/studio-cli

# Shell script
curl -fsSL https://tokens.studio/install.sh | sh
```
